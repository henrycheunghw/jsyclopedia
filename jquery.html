jQuery Concepts

jQuery Selector
Definition

jQuery selectors are used to select HTML elements based on their name, id, class, attribute, and more.

Syntax

The syntax for a jQuery selector is $(selector), where "$" is the shorthand for jQuery, and "selector" is the element(s) you want to select.

DOM: Get and Set
Definition

Get and Set refer to the process of getting or setting values or attributes of HTML elements using JavaScript's Document Object Model (DOM) methods.

Get Examples

document.getElementById("myInput").value

document.getElementById("myDiv").textContent

Set Examples

document.getElementById("myDiv").textContent = "New content"

document.getElementById("myImage").src = "new-image.jpg"
